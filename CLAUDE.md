# CLAUDE.md

## 项目简介
这是一个练手项目，用户需要完整开发此项目以学习了解SpringBoot与SpringAI的开发流程。

## 项目规划
这是一个用于存放RAG的站点，基于 Spring Boot 与 Spring AI 构建的 RAG-as-a-Service 平台。支持用户自主注册并上传文本文档，系统自动完成文档解析、文本分块（Chunking）、Embedding 生成及向量入库全流程，对外提供标准化 RESTful API，供第三方大模型应用即插即用地获取检索增强能力，免去自建 RAG 管线的成本。

## 回复规则
- 尽量引导用户学习标准开发流程与代码规范，如果用户偏离正确路线，提醒用户错误和正确的方式
